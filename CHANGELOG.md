# Change Log

- Added support for Font Awesome 4.7.0
- Added configuration option to select the Font Awesome version (4 or 5)
- **Breaking changes**
  - Renamed all configuration options from `fontAwesome5Autocomplete.*` to `fontAwesomeAutocomplete.*`
